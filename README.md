# SPA

To build the project:

1. Install node and npm
2. Execute next commands:
  ```
  npm install
  ```
  ```
  npm install webpack -g
  ```
  ```
  npm run build
  ```
3. To start the server run
  ```
  npm start
  ```
4. To run dev server:
  ```
  npm run dev-server
  ```
  
  
Project open on port 3000
